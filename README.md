# 💊 Landing Page - Drogaria

> Uma landing page moderna, limpa e de alta conversão desenvolvida para o segmento farmacêutico/drogaria. O projeto foi estruturado com foco em usabilidade, agilidade no carregamento e design responsivo.

---

## 📱 Demonstração

👉 [Acesse a Landing Page Online](https://joaoprochas.github.io/Template-Landing-Page-Drogaria/)

---

## ✨ Funcionalidades

*   **Design Clean & Profissional:** Cores e elementos visuais alinhados com a identidade visual do setor de saúde.
*   **Responsividade Total:** Interface adaptável para uma ótima experiência tanto em smartphones quanto em desktops.
*   **Seções Estruturadas:**
    *   **Hero Section:** Apresentação clara da drogaria.
    *   **Benefícios:** Vitrine para exibir as vantagens competitivas.
    *   **Localização e Contato:** Informações essenciais para o cliente encontrar a farmácia ou pedir pelo WhatsApp.
*   **Navegação Rápida:** Otimização para rolagem suave e carregamento ágil das seções.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

*   **HTML5:** Para a estruturação semântica da página.
*   **CSS3:** Para a estilização, layout responsivo e transições.
*   **JavaScript:** Para interatividades.

---

## 🚀 Como Executar Localmente

Caso queira clonar o repositório e rodar o projeto na sua máquina:

1. Clone o repositório: 
`bash
   git clone https://github.com/joaoprochas/Landing-Page-Drogaria.git`

2. Acesse a pasta do projeto:
`bash
    cd Landing-Page-Drogaria`
3. Abra o arquivo index.html no seu navegador de preferência ou utilize a extensão Live Server no VS Code.
